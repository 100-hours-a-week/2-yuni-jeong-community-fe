## 🌎 실행 방법
**1. git clone**
```bash
git clone https://github.com/100-hours-a-week/2-yuni-jeong-community-fe.git
```

**2. 의존성 설치**
```bash
npm install
```

**3. 실행**
```bash
npm run start
```

**4. 브라우저에서 열기**
```
http://localhost:3000
```
